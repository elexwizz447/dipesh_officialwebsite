WHAT: Numerical Investigation and Parametric Study of Subsonic Tandem-Axial Flow Compressor for Aerodynamic Optimization

My undergraduate thesis addresses high-load compressor design challenges by developing a design methodology and performance model for a subsonic tandem-axial flow rotor to enhance stage pressure ratio while mitigating adverse pressure gradients, flow separation, and end-wall losses. Objectives include creating 2D/3D blade geometries using parabolic arc methods, conducting CFD simulations with ANSYS Fluent to analyze flow physics, and performing parametric studies on pitch percentage (PP) and axial overlap (AO) to optimize performance at on/off-design conditions.

HOW: Systematic Design and Numerical Methodology

Developed Python code for the design of 2D tandem airfoils and 3D rotors with spanwise variations in stagger, chord, reaction, and diffusion factor using velocity triangle model and parabolic arc methods, and incorporating AO/PP for boundary layer control.
Developed propulsion performance model and implemented CFD study in ANSYS Fluent, including grid independence studies, design space exploration, and passage flow analysis
Developed Python scripts for parametric analysis, varying PP (70-80%) and AO (-1 to 5%), evaluating metrics like pressure ratio, efficiency, and loss coefficient at on/off-design conditions for design optmization

RESULTS: Optimized Performance and Flow Insights

Performance Gains
Tandem rotor achieved higher pressure rise,better efficiency, and lower loss coefficient than conventional design, attributed to gap nozzle flow energizing boundary layers and reducing separation.
Parametric Analysis
Optimal 80% PP and 1% AO configuration increased pressure rise and efficiency, with wider stall-free range and minimized hub separation.
Flow Physics and Contributions
Analysis revealed performance improvement at engine level with a reduction in TSFC due to enhanced boundary layer control, reduced adverse pressure gradients, and mitigated end-wall losses, providing a preliminary design philosophy and performance model for tandem rotors and a Python GUI for flow angle assessment.

Semi-Tapered Wing Planform for Regional Hybrid/Electric Aircraft

Linked aerodynamic findings to aircraft-level impacts on power demand, battery sizing, and hybrid system feasibility.

WHAT: Modal Analysis of Ariane Payload Fairing

This project involved a detailed modal analysis of the Ariane 5 medium payload fairing (half-model, aluminum shell, free-free boundary conditions) using NASTRAN/PATRAN to characterize natural frequencies and mode shapes. I conducted sensitivity studies on thickness doubling, honeycomb core integration, and I-beam stiffeners to tune the circumferential shell mode to a target 1 Hz frequency, balancing stiffness enhancements with mass constraints for structural integrity during separation. This work helped me improve my skills in finite element modeling, dynamic analysis, and optimization strategies, which are critical for aerospace vibration control. This project also made me that there are always clever approaches that you can come up with as a designer, if you understand from first principles the theoretical framework governing the dynamic behaviour of aerospace systems.

Fairing Mesh Overview

Detailed Mesh View

HOW: Finite Element Setup and Sensitivity Studies

Developed a half-model FE setup in PATRAN/NASTRAN with triangular shell elements (12800 elements) under free-free conditions, calculating baseline modes to identify low-frequency issues.
Conducted thickness sensitivity by doubling shell thickness, analyzing stiffness-mass trade-offs via eigenvalue solutions to evaluate frequency gains.
Integrated honeycomb core using equivalent properties, iterating core depth and plate thickness to optimize stiffness-to-mass ratio for target frequency.
Added I-beam stiffeners with non-equidistant placement targeting high-strain regions, assessing localized rigidity enhancements through multiple configurations.

RESULTS: Optimized Frequency Tuning and Trade-offs

Baseline and Thickness Doubling

WHAT: Finite Element Analysis of an Aircraft-Grade I-Beam

In this project, I analyzed a steel I-beam using MSC Nastran and Patran to understand how structural models capture deformation and stress under shear loading. I built both 1D and 2D finite element representations, compared them with analytical beam theories, and performed mesh convergence studies. This work went beyond running simulations—it helped me see how engineers balance computational efficiency with the need for local accuracy, and how structural modeling feeds into safety margins, material choices, and certification decisions in aerospace design.

I-Beam Cross-Section Geometry

1D Beam Model Setup

HOW: Modeling and Analysis Methodology

Constructed 1D beam model using CBEAM elements in Patran, incorporating Timoshenko theory to account for bending and shear deformations, with clamped conditions at one end and distributed shear load at the other.
Developed 2D shell model with CQUAD4 elements based on Mindlin-Reissner theory, enabling detailed capture of in-plane and out-of-plane behaviors, including membrane and shear effects.
Performed grid independence study, refining meshes to ensure solution stability, demonstrating how finer discretizations improve accuracy in local stress predictions while 1D models suffice for global responses.
Validated FEM outputs against analytical beam theories, highlighting how 1D approaches efficiently model global deflections while 2D captures localized phenomena like shear lag and flange warping.

RESULTS: Structural Insights and Aerospace Relevance

1D Beam Model
Captured global deflection efficiently, confirming theoretical values and showing how simple models remain powerful tools for preliminary design checks.
2D Shell Model
Revealed local stress patterns, shear effects, and flange warping, highlighting areas where detailed modeling is critical for durability and fatigue assessments.
Engineering Takeaways
Learned how model fidelity influences engineering decisions: when to prioritize speed vs. accuracy, how convergence builds confidence, and how FEM supports certification-level safety margins in aerospace structures.
